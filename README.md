## Quick start

- [Download from Github](https://github.com/themewagon/dnx/archive/refs/heads/main.zip) or clone the repo : `git clone https://github.com/themewagon/dnx.git`
- Recommended `Node.js v18.x`.
- **Install:** `npm install`
- **Start:** `npm run dev`
- **Build:** `npm run build`
